# Projet PFA – Ordonnancement industriel

## Contexte
Ce projet s’inscrit dans le cadre du Projet de Fin d’Année (PFA) en génie industriel.
Il porte sur l’optimisation de l’ordonnancement de la production dans un Flow Shop hybride
à machines dédiées, afin d’améliorer la performance globale du système de production.

## Problématique
Dans un environnement industriel complexe, l’ordonnancement des tâches sur des machines
dédiées constitue un enjeu majeur pour :
- réduire les temps de cycle,
- minimiser les retards,
- améliorer le respect des délais de livraison,
- optimiser l’utilisation des ressources.

## Objectifs
- Modéliser un problème d’ordonnancement Flow Shop hybride
- Proposer une méthode d’optimisation adaptée
- Comparer différentes règles d’ordonnancement
- Évaluer les performances à l’aide d’indicateurs clés

## Méthodologie
- Analyse du système de production
- Modélisation du Flow Shop hybride à machines dédiées
- Implémentation des règles d’ordonnancement
- Simulation et comparaison des performances
- Analyse des résultats

## Outils utilisés
- Python
- Excel
- Méthodes d’ordonnancement industriel
- Indicateurs de performance (Cmax, temps de cycle, retards)

## Résultats
- Amélioration de la performance de production
- Réduction des temps d’attente
- Meilleure utilisation des machines
